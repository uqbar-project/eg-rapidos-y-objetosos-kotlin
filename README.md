# Rápidos y objetosos

[![Build Status](https://travis-ci.org/uqbar-project/eg-rapidos-y-objetosos-kotlin.svg?branch=master)](https://travis-ci.org/uqbar-project/eg-rapidos-y-objetosos-kotlin)

## Descripción
Basados en el [enunciado de uno de los TP](https://docs.google.com/document/d/1m4zWUbiUPI-Ck_Kx9QXLWM9xSq425fpAhXgV6WHQjpc/edit#) y el ejemplo [combinando programación funcional y objetos](https://github.com/uqbar-project/eg-rapidos-y-objetosos-ts) implementado en typescript, dejamos una nueva solución cominando estas técnicas pero esta vez en implementado en [Kotlin](https://kotlinlang.org/). 

## Como correr el ejemplo
Como IDE vamos a utilizar [Intellij Idea](https://www.jetbrains.com/idea/) que por defecto tiene soporte del lenguaje. Una vez instalado el entorno y clonado el proyecto, solamente debemos abrir la carpeta donde descargamos el repositorio.

Para correr los tests podemos pararnos en el archivo /src/test/Kotlin/tests.kt y ejecutar desde el menú contextual **Run 'TestPilotos'**.
